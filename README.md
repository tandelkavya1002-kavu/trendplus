# TrendPulse
Real-Time Stock Sentiment Dashboard

## Features
- Real-time stock prices (TSLA, NVDA, AAPL, AMZN, MSFT)
- Sentiment Analysis using VADER
- Correlation Analysis
- Beautiful dark dashboard

## Tech Stack
- Python, FastAPI, VADER, yFinance, HTML/CSS

## How to Run
1. conda activate trendpulse
2. python -m uvicorn backend.main:app --reload
3. Open frontend/index.html
